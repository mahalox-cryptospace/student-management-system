# Student Management System (C++)

This is a simple C++ project that manages a list of students using only `main.cpp`.  
It includes basic operations such as adding a student, displaying information, and storing student data in memory.

## Features
- Add new students
- Store ID, name, GPA (AGNO) and graduation status
- Display list of students
- Uses structs, arrays, and basic functions in a single file
